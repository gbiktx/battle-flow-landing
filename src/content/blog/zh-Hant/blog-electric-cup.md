電屬性杯雖然在屬性上受到限制，但仍具有深層結構。根據我們的數據，電蜘蛛（Galvantula）位居榜首，但得分較高的陣容並非僅圍繞電蜘蛛構建。副屬性和角色壓縮比「全電屬性」標籤所暗示的要重要得多。

## Meta 一覽

由於只有 68 隻符合條件的寶可夢，這是一個競爭激烈的環境。既然大家都是電屬性，真正的 meta 是由**副屬性**決定的——蟲、幽靈、毒、格鬥、草、水、岩石和地面才是關鍵的戰場。

頂部的得分曲線非常陡峭：

| 等級               | 寶可夢                                                                        |
| ------------------ | ------------------------------------------------------------------------------ |
| **S 級 (90+)**     | 電蜘蛛、暗影電蜘蛛、洗翠頑皮雷彈、洛托姆、毒電嬰、電肚蛙 |
| **A 級 (80-90)**   | 正電拍拍、雷公、暗影雷電雲、巴布土撥、電燈怪、負電拍拍、割草洛托姆             |
| **B 級 (70-80)**   | 29 隻寶可夢壓縮在 10 分的區間內——這裡就是「非主流（spice）」存在的地方     |

第一名的電蜘蛛（得分 97.9）與第七名的正電拍拍（得分 88.3）之間有近 10 分的差距。對於任何賽制來說，這都是巨大的。

## 關鍵寶可夢

### 電蜘蛛 —— 頂級首發點（得分：97.9）

**蟲/電** | 連斬 + 撲擊 + 毒十字

這並非典型的電屬性攻擊者。電蜘蛛運行蟲屬性和毒屬性招式，而不是依賴電屬性特殊招式傷害，這使其在對戰覆蓋面上與大多數寶可夢截然不同。在我們的杯賽數據集中，它是**前 30 名寶可夢中 22 隻**的反制點，這就是為什麼大多數強勢陣容仍需要一個明確的針對電蜘蛛的計劃。

它的蟲屬性是關鍵——它抵抗格鬥和草屬性，同時對大多數電屬性招式受到的傷害為中性。唯一能真正封鎖它的是毒屬性（毒電嬰）和鋼屬性（自爆磁怪、托戈德瑪爾），因為兩者都抵抗它的整套招式。暗影電蜘蛛以坦度換取傷害，在陣容組合中表現通常優于常規版本。

### 洗翠頑皮雷彈 —— 草屬性通配符（得分：93.0）

**電/草** | 電擊 + 能量球 + 高速星星

能量球在這種賽制中既罕見又珍貴。它能重創電燈怪、阿羅拉隆隆岩和其他岩石屬性選擇。草屬性副屬性為洗翠頑皮雷彈提供了幾乎沒有其他人可以複製的特徵。電蜘蛛仍然能輕鬆處理它（評分 303），但洗翠頑皮雷彈出現在近一半的精英級陣容中。

### 毒電嬰 —— 電蜘蛛的答案（得分：91.0，評分：754）

**電/毒** | 溶解液 + 伏特攻擊 + 增強拳

整個賽制中最高的原始評分。毒屬性抵抗電蜘蛛的蟲屬性招式和毒十字，使毒電嬰成為杯賽中最乾淨俐落的反電蜘蛛選擇之一。當護盾情況合適時，增強拳使其具有真正的「滾雪球」潛力。

毒電嬰也是**該賽制中最常見的威脅**——它被視為 77% 精英級陣容的主要威脅。每個人都必須面對它，而只有少數寶可夢能乾淨俐落地處理它。注意地面屬性的覆蓋——擁有覺醒力量（地面）的倫琴貓可以摧毀它。

### 電燈怪 —— 高價值陣容組件（得分：86.2）

**水/電** | 水槍 + 衝浪 + 十萬伏特

儘管電燈怪的個人排名較低，但它是該賽制中最有價值的陣容構建組件之一。我們在相同的陣容框架中測試了電燈怪與電肚蛙，**在每一次檢查的對比中，電燈怪的表現都優於電肚蛙**，通常在整體陣容得分上高出 2-5 分。

為什麼？在全電屬性杯中，水屬性的覆蓋是獨特且極其珍貴的。衝浪能對岩石屬性（阿羅拉隆隆岩、阿羅拉隆隆石）造成效果絕佳的傷害——而岩石屬性是出現在 40% 以上精英陣容中的關鍵 meta 組件。電燈怪還能擊敗巴布土撥（評分 579），而電肚蛙則會被它農能（評分 200）。在 meta 前 10 名的選擇中，這有 +379 的巨大差距。

電燈怪的水/電雙屬性賦予了它額外的抗性和獨特的攻擊壓力，這是純電屬性寶可夢無法複製的。

### 洛托姆 —— 幽靈因素（得分：91.5）

**電/幽靈** | 驚嚇 + 十萬伏特 + 奇異之風

洛托姆的原始評分僅為 567，但其加權得分高達 91.5（總排名第 4）。它擊敗了那些關鍵的寶可夢，輸給了那些無關緊要的。幽靈屬性雙倍抵抗格鬥，使得巴布土撥的近身戰幾乎像撓癢癢一樣（評分 768——一場壓倒性的勝利）。它還擊敗了電蜘蛛（580）和洗翠頑皮雷彈（533）。

它的表親**割草洛托姆**（電/草，驚嚇 + 奇異之風 + 葉暴風）也值得一提——由於草屬性的覆蓋，它在大多數陣容框架中表現優於常規洛托姆，儘管常規洛托姆在某些包含洗翠頑皮雷彈的組合中更具優勢。

### 電肚蛙 —— 專家級選擇（得分：90.6）

**電** | 突襲 + 電磁炮 + 拋物面充電

擁有純粹的坦度，並透過快招突襲提供惡屬性覆蓋。強力反制毒電嬰（評分 636）和洛托姆（評分 665）——兩者都是 meta 前 5 名的選擇。關鍵招式細節是：**拋物面充電**保證電肚蛙獲得 +1 防禦增益，而**電磁炮**是高傷害的收割點（closer），只有 33% 的機會給對手施加 -1 攻擊減益。

電肚蛙在特定組合中表現出色——它支撐了兩個精英級陣容。但它是專家，而非通用選擇。它與巴布土撥的對戰（評分 200）是該賽制中最一邊倒的對戰之一。擁有電肚蛙的陣容需要另外兩名成員來處理巴布土撥和電蜘蛛。

### 巴布土撥 —— 壓力閥（得分：86.5）

**電/格鬥** | 踢倒 + 劈瓦 + 近身戰

全電屬性杯中的格鬥屬性。近身戰可以重創任何不抵抗它的寶可夢。摧毀電肚蛙、阿羅拉隆隆岩、電龍和大多數純電屬性。但幽靈屬性雙倍抵抗格鬥（洛托姆），蟲屬性抵抗它（電蜘蛛），毒屬性也抵抗它（毒電嬰）——因此頂級 meta 選擇都能處理它。

巴布土撥讓坦克型電屬性寶可夢不敢掉以輕心，它作為靈活的施壓組件出現在 25% 的精英陣容中。

### 阿羅拉隆隆岩 / 隆隆石 —— 岩石錨點

**岩石/電** | 滾動 + 尖石攻擊 + 岩爆（隆隆岩） | 落石 + 尖石攻擊 + 岩爆（隆隆石）

這些是該賽制中最有用的角色填充者。個人得分適中（阿羅拉隆隆岩得分 76.7，暗影隆隆石得分 75.7），但岩石屬性提供了重要的覆蓋——尖石攻擊重創電蜘蛛和飛行屬性，且岩石屬性抵抗電蜘蛛的蟲屬性招式。在我們的陣容結果中，它們合計出現在超過 40% 的精英陣容中。

## 反制鏈

Meta 形成了一個帶有支撐組件的四向循環：

```
電蜘蛛 → 統治大多數 meta
    ↓ 輸給
毒電嬰 → 封鎖電蜘蛛（毒屬性抵抗蟲 + 毒）
    ↓ 輸給
岩石屬性（阿羅拉隆隆岩/隆隆石） → 擊敗電蜘蛛和毒電嬰
    ↓ 輸給
電燈怪 / 巴布土撥 / 洗翠頑皮雷彈 → 全部擊敗岩石屬性
    ↓ 輸給
電蜘蛛 → 擊敗巴布土撥（蟲屬性抵抗格鬥）、
          洗翠頑皮雷彈（蟲屬性克制草屬性）、
          以及電燈怪（評分 360）
```

岩石屬性是關鍵連結。它們同時處理電蜘蛛和毒電嬰——兩個最大的威脅——這就是為什麼儘管個人得分在 70 分中段，它們仍出現在超過 40% 的精英陣容中。

**電肚蛙**作為毒電嬰 + 洛托姆的反制點切入，而**洛托姆**憑藉幽靈屬性創造的獨特對戰，遊離於循環之上。但核心 meta 衝突仍是：電蜘蛛 vs 毒電嬰 vs 岩石屬性 vs 水/格鬥/草屬性。

## 這是什麼樣的 Meta？

**中等 RPS（剪刀石頭布），有技術發揮的空間。**

RPS 的情況：

- 只有 68 隻寶可夢，頂級選擇定義了一切
- 存在硬反制：自爆磁怪 313 vs 電蜘蛛，巴布土撥 200 vs 電肚蛙
- 副屬性創造了僵硬的三角關係

技術的空間：

- 電蜘蛛的撲擊減益使得能量管理變得至關重要——你是留在場上吃攻擊下降，還是換人並放棄對位？
- 到處都是騙盾（bait）招式組合（毒電嬰的增強拳，巴布土撥的劈瓦）
- 洛托姆的幽靈屬性創造了獨特的中局決策
- 壓縮的 B 級（29 隻寶可夢得分在 70-80 範圍）意味著非主流（spice）選擇真的可以帶來驚喜

**坦度水平：** 混合型。電肚蛙和電燈怪是坦克，但暗影雷公和暗影電蜘蛛是玻璃大砲。不是一個超快節奏的賽制——有足夠的坦度讓真正的中局操作和能量管理發揮作用。

## 值得考慮的非主流選擇（Spice）

**暗影雷公**（得分 78.7，評分 746） —— 玻璃大砲。影子球 + 波導彈提供了該杯賽中獨特的幽靈和格鬥覆蓋。當與毒電嬰和岩石屬性搭配時，出現在多個精英陣容中。

**托戈德瑪爾**（電/鋼，得分 74.2） —— 鋼屬性封鎖了電蜘蛛的全套招式。致命針消耗低且能提升攻擊，而伏特攻擊賦予其真正的收割能力。它不如主流選擇常見，但它為陣容提供了一個真正的反電蜘蛛位。

**倫琴貓 / 暗影倫琴貓**（得分 73-74） —— 覺醒力量（地面）罕見且具有毀滅性。擊敗毒電嬰、電肚蛙、阿羅拉隆隆岩和電燈怪。要點是：覺醒力量（地面）是一場彩票——你需要一隻隨機到正確覺醒屬性的倫琴貓。

**暗影閃電鳥**（得分 74.3） —— 鑽嘴重創巴布土撥，熱風烘烤托戈德瑪爾和自爆磁怪。同時擊敗巴布土撥和洗翠頑皮雷彈。

**電擊魔獸**（得分 74.8） —— 踢倒 + 冰凍拳 + 噴火。極具多樣性的招式組合，無人能預料。冰凍拳是此杯賽中獨特的覆蓋。

## 陣容構建原則

1. **你必須有一個電蜘蛛的答案。** 毒電嬰、自爆磁怪、托戈德瑪爾或岩石屬性。沒有例外——它反制前 30 名中的 22 隻。

2. **你必須有一個巴布土撥的答案。** 洛托姆、電蜘蛛或毒電嬰。如果你沒有準備好，格鬥招式的連發將摧毀你的陣容。

3. **考慮副屬性，而非電屬性。** 每個人都是電屬性。真正的陣容構建謎題是平衡蟲、幽靈、毒、格鬥、草、水、岩石和地面的覆蓋。

4. **電燈怪作為陣容組件的表現優於其個人得分 (86.2) 所暗示的。** 在全電屬性賽制中，水屬性覆蓋是獨特的。它能處理岩石屬性，擊敗巴布土撥，並增加坦度。

5. **岩石屬性覆蓋至關重要。** 阿羅拉隆隆岩和隆隆石出現在極大比例的頂級評分陣容中。岩石屬性打擊電蜘蛛、飛行屬性，並提供獨特的防禦屬性。

## 前 5 名推薦陣容

我們使用 BattleFlow 的陣容生成器從前 30 名寶可夢中生成了 500 多套陣容，然後透過完整的評分引擎針對 50 個對手組成的池驗證了最佳組合。以下是前 5 名：

### 1. 洗翠頑皮雷彈 / 毒電嬰 / 阿羅拉隆隆岩 — BattleFlow: 92.4 (精英級)

| 維度           | 得分 |
| -------------- | ----- |
| 威脅控制       | 90.1  |
| 覆蓋面         | 97.1  |
| 恢復力         | 98.3  |
| 穩定性         | 91.8  |

我們目前電屬性杯運行中得分最高的陣容。洗翠頑皮雷彈首發（lead），利用草屬性覆蓋重創岩石屬性和電燈怪。毒電嬰作為安全切換點（safe swap）切入多種電蜘蛛對陣，並能很好地處理許多中性交換。阿羅拉隆隆岩作為收割點（closer），利用岩石傷害給電蜘蛛和飛行屬性施壓。核心在於結構：草、毒和岩石的覆蓋使得該三人組比大多數看起來只有電屬性的陣容具有更乾淨的分布。

### 2. 毒電嬰 / 電燈怪 / 割草洛托姆 — BattleFlow: 92.1 (精英級)

| 維度           | 得分 |
| -------------- | ----- |
| 威脅控制       | 89.9  |
| 覆蓋面         | 96.6  |
| 恢復力         | 97.8  |
| 穩定性         | 91.3  |

電燈怪再次證明了其價值。毒電嬰首發並乾脆俐落地壓制電蜘蛛。電燈怪作為帶有水屬性覆蓋的安全切換點——處理岩石屬性和巴布土撥，同時帶來足夠的坦度以在切換冷卻時間內保持有用。割草洛托姆作為收割點，提供草 + 幽靈覆蓋（葉暴風針對岩石屬性，奇異之風提供幽靈壓力）。三種不同的副屬性（毒、水、草/幽靈）賦予了這套陣容出色的屬性多樣性。

### 3. 暗影電蜘蛛 / 暗影雷公 / 托戈德瑪爾 — BattleFlow: 91.8 (精英級)

| 維度           | 得分 |
| -------------- | ----- |
| 威脅控制       | 89.8  |
| 覆蓋面         | 95.7  |
| 恢復力         | 97.7  |
| 穩定性         | 90.6  |

一套激進的陣容，以托戈德瑪爾作為陣容的防禦骨架。暗影電蜘蛛首發並施加最大的撲擊壓力——暗影加成使其打擊力度明顯強於常規版本。暗影雷公是通配符——影子球提供幽靈覆蓋，波導彈提供格鬥覆蓋。托戈德瑪爾作為抵抗電蜘蛛全套招式的鋼牆收尾，利用致命針提升攻擊，伏特攻擊進行收割。兩隻暗影攻擊者加上一隻覆蓋它們共同弱點的鋼屬性——這套陣容打擊有力，且對該賽制的頂級威脅有結構性的答案。

### 4. 洗翠頑皮雷彈 / 毒電嬰 / 阿羅拉隆隆石 — BattleFlow: 91.7 (精英級)

| 維度           | 得分 |

| 覆蓋面         | 97.0  |
| 恢復力         | 98.3  |
| 穩定性         | 91.8  |

第 1 名陣容的近親變體，將阿羅拉隆隆岩更換為隆隆石。權衡點在於：隆隆石利用落石打擊更重（暗影隆隆石更是如此），而阿羅拉隆隆岩透過滾動帶來更多坦度和能量獲取。陣容得分幾乎完全相同——97.0 的覆蓋面和 98.3 的恢復力與頂級陣容的精英維度相匹配。同樣的比賽計劃：洗翠頑皮雷彈首發草屬性，毒電嬰針對電蜘蛛，岩石屬性收尾。選擇你已經練好的那隻岩石寶可夢。

### 5. 洗翠頑皮雷彈 / 毒電嬰 / 電燈怪 — BattleFlow: 91.6 (精英級)

| 維度           | 得分 |

## 結語

電屬性杯的核心並不在於電屬性本身，而是在於你攜帶了哪些副屬性和覆蓋包。電蜘蛛雖然在個人排名中領先，但陣容數據表明，當你開始構建完整三人組時，電燈怪、岩石覆蓋和毒電嬰同樣重要。

我們分析中最大的發現：電燈怪（排名第 11，得分 86.2）在我們測試的每個陣容框架中表現都優於電肚蛙（排名第 6，得分 90.6）。全電屬性賽制中的水屬性覆蓋就是如此珍貴。同樣，阿羅拉隆隆岩和隆隆石（排名第 20-25，得分在 70 分中段）出現在精英陣容中的次數超過了大多數 S 級寶可夢，因為岩石覆蓋填補了一個關鍵空白。

以毒電嬰 + 電燈怪或毒电婴 + 岩石屬性作為你的核心，添加第三隻覆蓋它們共同弱點的寶可夢，你就能很好地應對天梯中的任何挑戰。
