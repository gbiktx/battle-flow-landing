电属性杯虽然在属性上受到限制，但仍具有深层结构。根据我们的数据，电蜘蛛（Galvantula）位居榜首，但得分较高的阵容并非仅围绕电蜘蛛构建。副属性和角色压缩比“全电属性”标签所暗示的要重要得多。

## Meta 一览

由于只有 68 只符合条件的宝可梦，这是一个竞争激烈的环境。既然大家都是电属性，真正的 meta 是由**副属性**决定的——虫、幽灵、毒、格斗、草、水、岩石和地面才是关键的战场。

顶部的得分曲线非常陡峭：

| 等级               | 宝可梦                                                                        |
| ------------------ | ------------------------------------------------------------------------------ |
| **S 级 (90+)**     | 电蜘蛛、暗影电蜘蛛、洗翠顽皮雷弹、洛托姆、毒电婴、电肚蛙 |
| **A 级 (80-90)**   | 正电拍拍、雷公、暗影雷电云、巴布土拨、电灯怪、负电拍拍、割草洛托姆             |
| **B 级 (70-80)**   | 29 只宝可梦压缩在 10 分的区间内——这里是“黑马（spice）”存在的地方     |

第一名的电蜘蛛（得分 97.9）与第七名的正电拍拍（得分 88.3）之间有近 10 分的差距。对于任何赛制来说，这都是巨大的。

## 关键宝可梦

### 电蜘蛛 —— 顶级首发点（得分：97.9）

**虫/电** | 连斩 + 扑击 + 毒十字

这并非典型的电属性攻击者。电蜘蛛运行虫属性和毒属性招式，而不是依赖电属性特殊招式伤害，这使其在对战覆盖面上与大多数宝可梦截然不同。在我们的杯赛数据集中，它是**前 30 名宝可梦中 22 只**的反制点，这就是为什么大多数强势阵容仍需要一个明确的针对电蜘蛛的计划。

它的虫属性是关键——它抵抗格斗和草属性，同时对大多数电属性招式受到的伤害为中性。唯一能真正封锁它的是毒属性（毒电婴）和钢属性（自爆磁怪、托戈德玛尔），因为两者都抵抗它的整套招式。暗影电蜘蛛以坦度换取伤害，在阵容组合中表现通常优于常规版本。

### 洗翠顽皮雷弹 —— 草属性通配符（得分：93.0）

**电/草** | 电击 + 能量球 + 高速星星

能量球在这种赛制中既罕见又珍贵。它能重创电灯怪、阿罗拉隆隆岩和其他岩石属性选择。草属性副属性为洗翠顽皮雷弹提供了几乎没有其他人可以复制的特征。电蜘蛛仍然能轻松处理它（评分 303），但洗翠顽皮雷弹出现在近一半的精英级阵容中。

### 毒电婴 —— 电蜘蛛的答案（得分：91.0，评分：754）

**电/毒** | 溶解液 + 伏特攻击 + 增强拳

整个赛制中最高的原始评分。毒属性抵抗电蜘蛛的虫属性招式和毒十字，使毒电婴成为杯赛中最干净利落的反电蜘蛛选择之一。当护盾情况合适时，增强拳使其具有真正的“滚雪球”潜力。

毒电婴也是**该赛制中最常见的威胁**——它被视为 77% 精英级阵容的主要威胁。每个人都必须面对它，而只有少数宝可梦能干净利落地处理它。注意地面属性的覆盖——拥有觉醒力量（地面）的伦琴猫可以摧毁它。

### 电灯怪 —— 高价值阵容组件（得分：86.2）

**水/电** | 水枪 + 冲浪 + 十万伏特

尽管电灯怪的个人排名较低，但它是该赛制中最有价值的阵容构建组件之一。我们在相同的阵容框架中测试了电灯怪与电肚蛙，**在每一次检查的对比中，电灯怪的表现都优于电肚蛙**，通常在整体阵容得分上高出 2-5 分。

为什么？在全电属性杯中，水属性的覆盖是独特且极其珍贵的。冲浪能对岩石属性（阿罗拉隆隆岩、阿罗拉隆隆石）造成效果绝佳的伤害——而岩石属性是出现在 40% 以上精英阵容中的关键 meta 组件。电灯怪还能击败巴布土拨（评分 579），而电肚蛙则会被它农能（评分 200）。在 meta 前 10 名的选择中，这有 +379 的巨大差距。

电灯怪的水/电双属性赋予了它额外的抗性和独特的攻击压力，这是纯电属性宝可梦无法复制的。

### 洛托姆 —— 幽灵因素（得分：91.5）

**电/幽灵** | 惊吓 + 十万伏特 + 奇异之风

洛托姆的原始评分仅为 567，但其加权得分高达 91.5（总排名第 4）。它击败了那些关键的宝可梦，输给了那些无关紧要的。幽灵属性双倍抵抗格斗，使得巴布土拨的近身战几乎像挠痒痒一样（评分 768——一场压倒性的胜利）。它还击败了电蜘蛛（580）和洗翠顽皮雷弹（533）。

它的表亲**割草洛托姆**（电/草，惊吓 + 奇异之风 + 叶暴风）也值得一提——由于草属性的覆盖，它在大多数阵容框架中表现优于常规洛托姆，尽管常规洛托姆在某些包含洗翠顽皮雷弹的组合中更具优势。

### 电肚蛙 —— 专家级选择（得分：90.6）

**电** | 突袭 + 电磁炮 + 抛物面充电

拥有纯粹的坦度，并通过快招突袭提供恶属性覆盖。强力反制毒电婴（评分 636）和洛托姆（评分 665）——两者都是 meta 前 5 名的选择。关键招式细节是：**抛物面充电**保证电肚蛙获得 +1 防御增益，而**电磁炮**是高伤害的收割点（closer），只有 33% 的机会给对手施加 -1 攻击减益。

电肚蛙在特定组合中表现出色——它支撑了两个精英级阵容。但它是专家，而非通用选择。它与巴布土拨的对战（评分 200）是该赛制中最一边倒的对战之一。拥有电肚蛙的阵容需要另外两名成员来处理巴布土拨和电蜘蛛。

### 巴布土拨 —— 压力阀（得分：86.5）

**电/格斗** | 踢倒 + 劈瓦 + 近身战

全电属性杯中的格斗属性。近身战可以重创任何不抵抗它的宝可梦。摧毁电肚蛙、阿罗拉隆隆岩、电龙和大多数纯电属性。但幽灵属性双倍抵抗格斗（洛托姆），虫属性抵抗它（电蜘蛛），毒属性也抵抗它（毒电婴）——因此顶级 meta 选择都能处理它。

巴布土拨让坦克型电属性宝可梦不敢掉以轻心，它作为灵活的施压组件出现在 25% 的精英阵容中。

### 阿罗拉隆隆岩 / 隆隆石 —— 岩石锚点

**岩石/电** | 滚动 + 尖石攻击 + 岩爆（隆隆岩） | 落石 + 尖石攻击 + 岩爆（隆隆石）

这些是该赛制中最有用的角色填充者。个人得分适中（阿罗拉隆隆岩得分 76.7，暗影隆隆石得分 75.7），但岩石属性提供了重要的覆盖——尖石攻击重创电蜘蛛和飞行属性，且岩石属性抵抗电蜘蛛的虫属性招式。在我们的阵容结果中，它们合计出现在超过 40% 的精英阵容中。

## 反制链

Meta 形成了一个带有支撑组件的四向循环：

```
电蜘蛛 → 统治大多数 meta
    ↓ 输给
毒电婴 → 封锁电蜘蛛（毒属性抵抗虫 + 毒）
    ↓ 输给
岩石属性（阿罗拉隆隆岩/隆隆石） → 击败电蜘蛛和毒电婴
    ↓ 输给
电灯怪 / 巴布土拨 / 洗翠顽皮雷弹 → 全部击败岩石属性
    ↓ 输给
电蜘蛛 → 击败巴布土拨（虫属性抵抗格斗）、
          洗翠顽皮雷弹（虫属性克制草属性）、
          以及电灯怪（评分 360）
```

岩石属性是关键链接。它们同时处理电蜘蛛和毒电婴——两个最大的威胁——这就是为什么尽管个人得分在 70 分中段，它们仍出现在超过 40% 的精英阵容中。

**电肚蛙**作为毒电婴 + 洛托姆的反制点切入，而**洛托姆**凭借幽灵属性创造的独特对战，游离于循环之上。但核心 meta 冲突仍是：电蜘蛛 vs 毒电婴 vs 岩石属性 vs 水/格斗/草属性。

## 这是什么样的 Meta？

**中等 RPS（石头剪刀布），有技术发挥的空间。**

RPS 的情况：

- 只有 68 只宝可梦，顶级选择定义了一切
- 存在硬反制：自爆磁怪 313 vs 电蜘蛛，巴布土拨 200 vs 电肚蛙
- 副属性创造了僵硬的三角关系

技术的空间：

- 电蜘蛛的扑击减益使得能量管理变得至关重要——你是留在场上吃攻击下降，还是换人并放弃对位？
- 到处都是骗盾（bait）招式组合（毒电婴的增强拳，巴布土拨的劈瓦）
- 洛托姆的幽灵属性创造了独特的中局决策
- 压缩的 B 级（29 只宝可梦得分在 70-80 范围）意味着黑马（spice）选择真的可以带来惊喜

**坦度水平：** 混合型。电肚蛙和电灯怪是坦克，但暗影雷公和暗影电蜘蛛是玻璃大炮。不是一个超快节奏的赛制——有足够的坦度让真正的中局操作和能量管理发挥作用。

## 值得考虑的黑马选择（Spice）

**暗影雷公**（得分 78.7，评分 746） —— 玻璃大炮。影子球 + 波导弹提供了该杯赛中独特的幽灵和格斗覆盖。当与毒电婴和岩石属性搭配时，出现在多个精英阵容中。

**托戈德玛尔**（电/钢，得分 74.2） —— 钢属性封锁了电蜘蛛的全套招式。致命针消耗低且能提升攻击，而伏特攻击赋予其真正的收割能力。它不如主流选择常见，但它为阵容提供了一个真正的反电蜘蛛位。

**伦琴猫 / 暗影伦琴猫**（得分 73-74） —— 觉醒力量（地面）罕见且具有毁灭性。击败毒电婴、电肚蛙、阿罗拉隆隆岩和电灯怪。要点是：觉醒力量（地面）是一场彩票——你需要一只随机到正确觉醒属性的伦琴猫。

**暗影闪电鸟**（得分 74.3） —— 钻嘴重创巴布土拨，热风烘烤托戈德玛尔和自爆磁怪。同时击败巴布土拨和洗翠顽皮雷弹。

**电击魔兽**（得分 74.8） —— 踢倒 + 冰冻拳 + 喷火。极具多样性的招式组合，无人能预料。冰冻拳是此杯赛中独特的覆盖。

## 阵容构建原则

1. **你必须有一个电蜘蛛的答案。** 毒电婴、自爆磁怪、托戈德玛尔或岩石属性。没有例外——它反制前 30 名中的 22 只。

2. **你必须有一个巴布土拨的答案。** 洛托姆、电蜘蛛或毒电婴。如果你没有准备好，格斗招式的连发将摧毁你的阵容。

3. **考虑副属性，而非电属性。** 每个人都是电属性。真正的阵容构建谜题是平衡虫、幽灵、毒、格斗、草、水、岩石和地面的覆盖。

4. **电灯怪作为阵容组件的表现优于其个人得分 (86.2) 所暗示的。** 在全电属性赛制中，水属性覆盖是独特的。它能处理岩石属性，击败巴布土拨，并增加坦度。

5. **岩石属性覆盖至关重要。** 阿罗拉隆隆岩和隆隆石出现在极大比例的顶级评分阵容中。岩石属性打击电蜘蛛、飞行属性，并提供独特的防御属性。

## 前 5 名推荐阵容

我们使用 BattleFlow 的阵容生成器从前 30 名宝可梦中生成了 500 多套阵容，然后通过完整的评分引擎针对 50 个对手组成的池验证了最佳组合。以下是前 5 名：

### 1. 洗翠顽皮雷弹 / 毒电婴 / 阿罗拉隆隆岩 — BattleFlow: 92.4 (精英级)

| 维度           | 得分 |
| -------------- | ----- |
| 威胁控制       | 90.1  |
| 覆盖面         | 97.1  |
| 恢复力         | 98.3  |
| 稳定性         | 91.8  |

我们目前电属性杯运行中得分最高的阵容。洗翠顽皮雷弹首发（lead），利用草属性覆盖重创岩石属性和电灯怪。毒电婴作为安全切换点（safe swap）切入多种电蜘蛛对阵，并能很好地处理许多中性交换。阿罗拉隆隆岩作为收割点（closer），利用岩石伤害给电蜘蛛和飞行属性施压。核心在于结构：草、毒和岩石的覆盖使得该三人组比大多数看起来只有电属性的阵容具有更干净的分布。

### 2. 毒电婴 / 电灯怪 / 割草洛托姆 — BattleFlow: 92.1 (精英级)

| 覆盖面         | 96.6  |
| 恢复力         | 97.8  |
| 稳定性         | 91.3  |

电灯怪再次证明了其价值。毒电婴首发并干净利落地压制电蜘蛛。电灯怪作为带有水属性覆盖的安全切换点——处理岩石属性和巴布土拨，同时带来足够的坦度以在切换冷却时间内保持有用。割草洛托姆作为收割点，提供草 + 幽灵覆盖（叶暴风针对岩石属性，奇异之风提供幽灵压力）。三种不同的副属性（毒、水、草/幽灵）赋予了这套阵容出色的属性多样性。

### 3. 暗影电蜘蛛 / 暗影雷公 / 托戈德玛尔 — BattleFlow: 91.8 (精英级)

一套激进的阵容，以托戈德玛尔作为阵容的防御骨架。暗影电蜘蛛首发并施加最大的扑击压力——暗影加成使其打击力度明显强于常规版本。暗影雷公是通配符——影子球提供幽灵覆盖，波导弹提供格斗覆盖。托戈德玛尔作为抵抗电蜘蛛全套招式的钢墙收尾，利用致命针提升攻击，伏特攻击进行收割。两只暗影攻击者加上一只覆盖它们共同弱点的钢属性——这套阵容打击有力，且对该赛制的顶级威胁有结构性的答案。

### 4. 洗翠顽皮雷弹 / 毒电婴 / 阿罗拉隆隆石 — BattleFlow: 91.7 (精英级)

| 维度           | 得分 |

第 1 名阵容的近亲变体，将阿罗拉隆隆岩更换为隆隆石。权衡点在于：隆隆石利用落石打击更重（暗影隆隆石更是如此），而阿罗拉隆隆岩通过滚动带来更多坦度和能量获取。阵容得分几乎完全相同——97.0 的覆盖面和 98.3 的恢复力与顶级阵容的精英维度相匹配。同样的比赛计划：洗翠顽皮雷弹首发草属性，毒电婴针对电蜘蛛，岩石属性收尾。选择你已经练好的那只岩石宝可梦。

### 5. 洗翠顽皮雷弹 / 毒电婴 / 电灯怪 — BattleFlow: 91.6 (精英级)

| 维度           | 得分 |
| -------------- | ----- |

| 覆盖面         | 97.8  |
| 恢复力         | 98.1  |

此名单中覆盖面 (97.8) 和稳定性 (92.6) 最高的阵容。洗翠顽皮雷弹首发利用草属性覆盖重创岩石属性和电灯怪。毒电婴作为安全切换点——封锁电蜘蛛，利用增强拳滚雪球，并处理大多数中性对阵。电灯怪利用针对岩石属性和巴布土拨的水属性覆盖收尾。97.8 的覆盖面意味着几乎没有盲点，98.1 的恢复力意味着这套阵容几乎永远不会陷入必败之地。威胁控制略低于顶级选择，但其稳定性使其成为上分最稳妥的选择。

**同为 91.6 分：** 暗影电蜘蛛 / 洗翠顽皮雷弹 / 暗影隆隆石 (T:90.2 C:94.5 R:97.5 S:89.0) —— 经典的虫 + 草 + 岩石三角，利用暗影攻击者提升伤害上限。

## 结语

电属性杯的核心并不在于电属性本身，而在于你携带了哪些副属性和覆盖包。电蜘蛛虽然在个人排名中领先，但阵容数据表明，当你开始构建完整三人组时，电灯怪、岩石覆盖和毒电婴同样重要。

我们分析中最大的发现：电灯怪（排名第 11，得分 86.2）在我们测试的每个阵容框架中表现都优于电肚蛙（排名第 6，得分 90.6）。全电属性赛制中的水属性覆盖就是如此珍贵。同样，阿罗拉隆隆岩和隆隆石（排名第 20-25，得分在 70 分中段）出现在精英阵容中的次数超过了大多数 S 级宝可梦，因为岩石覆盖填补了一个关键空白。

以毒电婴 + 电灯怪或毒电婴 + 岩石属性作为你的核心，添加第三只覆盖它们共同弱点的宝可梦，你就能很好地应对天梯中的任何挑战。
